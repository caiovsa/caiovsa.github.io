# Caio Vasconcelos | AI Engineer & Data Scientist

Portfolio built with React, Vite, TypeScript, and shadcn/ui.

## Technologies

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion
- **Language**: TypeScript

## Development

1. Install dependencies:
   ```sh
   npm install
   ```

2. Run development server:
   ```sh
   npm run dev
   ```

3. Build for production:
   ```sh
   npm run build
   ```
